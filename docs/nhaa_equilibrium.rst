==========================
 NhaA equilibrium dataset
==========================

.. module:: MDAnalysisData.nhaa_equilibrium

The :mod:`MDAnalysisData.nhaa_equilibrium` module contains a 
500-ns equilibrium MD trajectory of a membrane-protein system
with all solvent removed. It can be used to study membrane-
protein dynamics.
